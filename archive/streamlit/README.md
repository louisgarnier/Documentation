# Streamlit Application

This is the original Streamlit version of the Test Case Documentation Tool.

## Running the Application

```bash
# From the project root
cd streamlit
streamlit run app.py
```

Or from the project root:

```bash
streamlit run streamlit/app.py
```

## Dependencies

Install dependencies from `requirements.txt`:

```bash
pip install -r requirements.txt
```

## Notes

- This version uses shared components from `../shared/`
- Database is located in `../shared/database/`
- Uploads are stored in `../uploads/` (relative to project root)
