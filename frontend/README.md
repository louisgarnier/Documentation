This frontend will provide a modern, user-friendly interface matching the design from `test-case-manager/`.

## Technology

- **Framework**: Next.js or Nuxt.js (TBD)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (matching test-case-manager design)

## Features

- Test case list with card-based layout
- Test case detail view
- Create/edit/delete operations
- Screenshot management
- Excel export
- Modern, responsive UI

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Design Reference

The design is based on the React components in `../test-case-manager/`:
